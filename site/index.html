System Capabilities

Lightweight C++

Written in pure C++ for minimal memory footprint and high performance.

Background Mode

Runs silently in the background without a visible window.

Full Key Support

Captures all alphanumeric keys, special characters, and system keys.

Local Storage

Logs are securely appended to a local text file for easy access.
